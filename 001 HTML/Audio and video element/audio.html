This Page will walk through you to show that how we can embed the audios in the web pages and I will try to give the description of some of the element and Attribute of audio element.

Element and Attribute Used

Description of list of all the element and attributes and why we are using them

Elements

<audio>
This is the most basic element for emebeding the audios in to the web pages
<figure>
This is a block level element used to put the images/audios/or other type and we can provide a specific caption to this audio using another element used inside the <figure> element is <figurecaption>
<figurecaption>
This element is used to provide the caption for the anything.
Mozilla Documentation says that the <figurecaption> element can be used to wrap any kind of emotion or text or code in which we want to bind the caption/title with it.

Attribute

<src>
This will provide the source for the Audio. where to look?
<controls>
This will provide the controls for the audio
<loop>
once the audio is completed wether we want to repeate it or not.
<muted>
The audio by default will be muted.
<autoplay>
once we clicked the page and audio will start to play without our permission.
<title>
This will provide title to the image when we hover over the image we see it .
<preload>
This will make sure that the audio will be preloaded before playing or once we are in the website.

And there are many more but this much of elements ans attribute will work for now.
