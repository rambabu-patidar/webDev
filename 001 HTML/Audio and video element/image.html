Element and Attribute Used

Description of list of all the element and attributes and why we are using them

Elements

<img>
This is the most basic element for emebeding the images in to the web pages
<figure>
This is a block level element used to put the images and we can provide a specific caption to this image using another element used inside the <figure> element is <figurecaption>
<figurecaption>
This element is used to provide the caption for the images.
Mozilla Documentation says that the <figurecaption> element can be used to wrap any kind of emotion or text or code in which we want to bind the caption with it.

Attribute

<src>
This will provide the source for the image. where to look.
<alt>
This will provide alternate text for the image if failde to load by any reasons
<width>
This will provide initial width to the image we have provided.
<height>
This will provide initial height of the imgae we have provided.
<loading>
This will only load the images when we are near the imgae otherwise it will not until it.
<title>
This will provide title to the image when we hover over the image we see it .
